BambuNRS — DIY Nozzle Repair Station

A fully open-source, dedicated workstation for testing, cleaning and maintaining Bambu Lab 3D printer nozzles — without tying up your printer.

Why BambuNRS?

- Keep your 3D printer running — perform nozzle maintenance on a separate platform
- More control — stable base lets you apply proper force for hot/cold pull procedures
- Full fan control — freely toggle hotend cooling fan during maintenance

Features:

- Hot/Cold pull maintenance for nozzle cleaning and replacement
- Heat hex wrench unclogging for heat creep clogs
- Cold pull for thorough hotend cleaning after any unclogging procedure
- PID temperature control with configurable parameters via Bluetooth
- Real-time OLED display showing actual and target temperatures (P.V. / S.V.)
- Thermal resistor open-circuit detection (displays "NC")
- Automatic cooling fan control with manual override
- Dual nozzle support: A/H series (A1, A1 mini, H2S, H2D, P2) and P/X series (P1, X1E)

What You Get:

- Complete PCB design files (Gerber, BOM, Pick & Place)
- CNC machining drawings for aluminum nozzle bases
- Arduino/ESP32 firmware with full source code
- Comprehensive documentation and assembly guide with photos
- 3D printable enclosure on MakerWorld

Specifications:

- Power: 24V DC input
- Temperature range: 50°C - 300°C
- Communication: Bluetooth serial for parameter configuration
- Three hardware configurations: AP Universal / A-Type / P-Type
